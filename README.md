# 🏡 Real Estate Data Analysis Project

## 📌 Overview
This project analyzes real estate data across major Indian cities to uncover pricing trends, demand hotspots, and investment opportunities.

## 🎯 Objective
- Identify price distribution patterns
- Detect high-demand locations
- Perform clustering for market segmentation

## 📊 Dataset
- Merged dataset of multiple cities
- Includes pricing, location, and property features

## 🔍 Analysis Performed
- Data Cleaning (handling missing values, formatting)
- Exploratory Data Analysis (EDA)
- Heatmaps for demand visualization
- Clustering (K-Means)
- Price valuation modeling

## 📈 Key Insights
- Major metro cities show clustered high-value zones
- Demand hotspots vary significantly across cities
- Certain regions show undervalued investment potential

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Folium

## 📁 Project Structure
- `DataCleaning.ipynb`
- `RE_Model.ipynb`
- `maps_output/`
- `merged_files.csv`

## 📌 Future Improvements
- Build interactive dashboard (Streamlit)
- Add real-time data pipeline
## 💡 Business Impact
- Identified high-demand real estate zones across major cities
- Highlighted undervalued areas for investment opportunities
- Enabled data-driven decision-making using clustering and heatmaps
